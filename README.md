## Tech Stack Used

- **Programming Language:** Kotlin
- **UI Toolkit:** Jetpack Compose
- **Data Retrieval:** Retrofit
- **Local Database:** Room
- **Dependency Injection:** Dagger or Hilt
- **Design Language:** Material Design principles
- **API:** https://www.alphavantage.co
